# Online-Voting-System-Secure-Web-Platform

A secure web-based voting system developed using HTML, CSS, JavaScript, Java, and MySQL. The system provides admin and voter authentication, prevents duplicate voting, and supports real-time vote counting through a web interface.

# Features
- Admin and voter login
- Secure vote casting (one vote per voter)
- Admin dashboard to manage candidates and results
- Session handling to prevent duplicate voting
- Real-time result computation

# Technologies Used
- HTML, CSS, JavaScript
- Java 
- MySQL
